What I do

Web Development

Most of my projects has included some sort of frontend web development. It's because I have mainly focused on the web, and built with Electron.

Technologies

I use Git and Github almost every day where I upload my different projects. I have a basic understanding of the NPM and Docker environment.

Databases

I have used both SQL and NoSQL (Non SQL) databases in my projects, where I have a basic understanding of the SQL language and MySQL structuring.

Web Libraries

Other Development

Backend Development
